# Project Overview

The HRIS implementation is designed as a multi-phase rollout to ensure controlled adoption, minimize risk, and deliver incremental business value.

The system focuses on end-to-end employee lifecycle management, covering processes from onboarding to exit, with an emphasis on automation, data centralization, and improved user experience.

# Phase-wise Implementation Plan

## Phase 1: Core HR Operations (MVP Release)

Modules Included:

A. Onboarding Management (Post-Selection)

Scope:
* Capture new employee details - Through e-forms
* Document submission & verification - Approval flow
* Role/department allocation - Information Dashboard
* System access initiation - Notification Management
  
Key Features:
* Structured onboarding workflow - data capturing e-forms
* Checklist-based task tracking dashboard
* HR visibility into onboarding status - systematic and automated notification triggers 
  
B. Employee Leave Management System

Scope:
* Leave application & tracking
* Multiple leave types (Sick, Casual, Earned, etc.)
* Approval workflows
* Leave balance visibility - dashboard
  
Key Features:
* Automated approval flow (Employee → Manager)
* Real-time leave balance validation
* Leave history tracking
* Employee dashboard view
  
### Phase 1 Objective:
* Digitize high-frequency HR operation - leave management
* Reduce manual dependency on HR - Onboarding data and workflow management
* Establish core system adoption - Data and process optimization

## Phase 2: Employee Self-Service & Information Hub

Modules Included:

A. HR Policies & Benefits
Scope:
* Centralized access to company policies - News and Updates Web Page
* Benefits information (insurance, allowances, etc.) - Employee Benefits Dashboard
* Employee Data Update e-Forms 

Forms Included:
1. Bank Details Change
2. Address Change
3. Benefits Application
4. General Query Submission
   
Key Features:
* Form submission tracking - database update
* Approval/processing workflows - automated notification and tasks for process flow
* Status notifications - Confirmation message triggers
  
B. Information Dashboard - Centralised Company News and Information viewing webpage
Key Features:
* Company announcements
* Policy updates
* Organizational news
* Important alerts
  
### Phase 2 Objective:
* Enable employee self-service - intuitive e-forms and automated workflows
* Improve communication transparency - Company-wide accessible webpage/dashboards
* Reduce HR operational workload 
  
## Phase 3: Advanced HR Lifecycle Management

Modules Included:

A. Leave of Absence (LOA) / Maternity
Scope and key features:
* Extended leave requests - e-forms and automated approval flows
* Policy-based validation - detailed information articles and validation pop-ups for employees' understanding
* Approval workflows - automated tasks and notifications

B. Appraisal & Promotion
Scope and key Features:
* Performance tracking dashboards - for employees and managers
* Promotion workflows - automated workflows
* Manager inputs & approvals - employee encouragement portals
  
C. Exit Management
Scope and key features:
* Resignation submission - e-form
* Exit clearance workflow - database update
* Knowledge transfer tracking - Task Dashboard
* Final settlement process - automated notification triggers
  
### Phase 3 Objective:
* Cover complete employee lifecycle
* Enable strategic HR decision-making
* Improve governance and compliance
