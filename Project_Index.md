# Project Index – HRIS Case Study Project (Real-World Simulation)

---

## Overview

This repository contains a **self-initiated HRIS (Human Resource Information System) case study project**, designed to simulate a real-world implementation.

The project focuses on **employee lifecycle management**, covering onboarding, leave management, and advanced HR processes through a **multi-phase rollout approach**.

---

## Project Objective

* Analyze existing HR processes (AS-IS)
* Identify gaps and inefficiencies
* Design optimized system-driven workflows (TO-BE)
* Define business and functional requirements
* Simulate real-world BA and QA deliverables
